# Armory Intendant

## Запуск
```bash
npm install
npm run dev
```
После запуска Vite откроет локальный адрес вида `http://localhost:5173`.

## Структура проекта
- `src/config/ScebesConfig.ts` — cписок сцен.
- `src/config/GameConfig.ts` — конфигурация Phaser.
- `src/scenes/BootScene.ts` — экран загрузки.
